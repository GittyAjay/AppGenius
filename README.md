# Avishkar.Ai App Generator

Welcome to Avishkar.Ai App Generator using chat window! This project allows you to generate any application based on your requirements. Simply follow the steps below to get started:

## Prerequisites

- Node.js installed on your machine.
- React Native setup on your system. If you haven't set up React Native yet, follow the instructions [here](https://reactnative.dev/docs/environment-setup).


## Setup

1. Clone this repository to your local machine:

   ```
   git clone https://github.com/GittyAjay/Avishkar.Ai.git
   ```

2. Navigate to the `react-component-generator` directory:

   ```
   cd avishkar-ai/react-component-generator
   ```
3. Run

   ```
   npm install
   ```   

4. Start the Node.js server:

   ```
   node index.js
   ```

## Generating a Chat App

1. Once the Node.js server is running, open your web browser and go to `http://localhost:3000`.

2. You will see a chat window. Enter your requirements for the application.

3. After providing your requirements, you will receive a message in the chat window with a download button.

4. Click on the download button to download the generated React Native chat application.

## Running app locally

1. Android
   ```
   npx react-native run-android
   ```

2. Ios

   ```
   npx pod-install
   npx react-native run-ios
   ```
